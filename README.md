# Descripción
Con este primer proyecto, se busca implementar una solución para el problema de planificación de tareas

# Instrucciones de cómo ejecutarlo

1. Crear un entorno virtual.
    python -m venv .venv
2. Activación.
    Para windows: .venv\Scripts\Activate.ps1
    para mac: source .venv/bin/activate
3. Instalar dependencias
    pip install -r requirements.txt
4. Ejecutar el programa
    python main.py <makespan_objetivo>
